Known Issues
  • Your snap will be rejected if it is not in the format Wing-Edit version 1.3.1 will save it. Error messages are more helpful now
  • Direct routing from stereo Busses to OUTs will either display incorrect or will lead to rejecting your SNAP altogether fixed
  • Direct Inputs on Matrixes are not yet rendered.
  • To report further issues or would like to provide your snap file for further extensions of this tool, please post in this forum thread or create a Github issue
